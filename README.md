## Dotfiles

### zsh

1. Install `zsh`
2. Install [Antigen](https://github.com/zsh-users/antigen)
3. Set `zsh` as default: `chsh -s $(which zsh)`
4. Copy `.zshrc`

### vim

1. Install `vim`
2. Install [Vundle](https://github.com/VundleVim/Vundle.vim)
3. Copy `.vimrc`
4. Install plugins `vim +PluginInstall +qall`
